## 📊 What is Lot Size in Forex?

Lot size refers to the volume of your trade.  
It determines how much you gain or lose per pip movement.

### Standard Types:
- Standard Lot = 100,000 units  
- Mini Lot = 10,000 units  
- Micro Lot = 1,000 units  

---

## 🧮 How to Calculate Lot Size

To calculate lot size, you need:

- Account balance  
- Risk percentage  
- Stop loss (pips)  
- Pair value  

### Basic Formula:
Lot Size = (Account Balance × Risk %) ÷ Stop Loss

---

## 💡 Example Calculation

- Balance: $1000  
- Risk: 1%  
- Stop Loss: 50 pips  

Result:  
You should trade around **0.02 lots**
